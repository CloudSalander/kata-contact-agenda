# kata-contact-agenda

Kata 68 per l'especialitat fullstackPHP 27-11-25

Es tracta de crear una agenda de contactes.

Cada contacte té un nom, cognoms, un nº de telèfon i una adreça de correu electrònic.

Hem de poder:

- Afegir,eliminar buscar i llistar contactes a l'agenda.
- Exportar informació de l'agenda en un fitxer.
